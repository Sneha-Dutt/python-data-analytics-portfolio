# 🎬 Netflix Content Analytics using Python & Pandas

## 📌 Project Overview
This project analyzes Netflix's global content catalog using Python and Pandas to uncover trends in content growth, country-wise distribution, genre popularity, audience targeting, and release patterns.

The objective was to demonstrate strong analytical thinking, business KPI design, and hands-on expertise in Pandas for real-world data analysis.

---

## 🎯 Business Questions Solved
- How has Netflix content grown over time?
- What is the ratio of Movies vs TV Shows?
- Which countries contribute the most titles?
- What are the most popular genres?
- What audience ratings dominate the platform?
- Are there seasonal release trends?
- What is the typical movie duration?

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- Matplotlib
- Google Colab
- Git & GitHub

---

## 📊 Key Insights
- Netflix catalog is movie-dominant (~69% Movies).
- Content growth peaked around 2019.
- United States leads content production, followed by India.
- Drama and International content are highly prevalent.
- TV-MA and TV-14 ratings dominate the platform.
- Most movies fall in the 80–120 minute range.
